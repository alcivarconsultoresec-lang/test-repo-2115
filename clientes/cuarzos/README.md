# Proyecto prospecto: Cuarzos Boutique

Demo comercial para prospecto que vende cuarzos, minerales y productos energéticos.

## URL esperada

```txt
https://alcivarconsultoresec-lang.github.io/test-repo-2115/clientes/cuarzos/
```

## Objetivo comercial

Convertir publicaciones dispersas o fotos sueltas de cuarzos en una vitrina boutique con:

- catálogo por intención,
- productos destacados,
- packs sugeridos,
- guía rápida para elegir cuarzo,
- botones WhatsApp por producto,
- enfoque de compra directa.

## Datos pendientes del prospecto

Reemplazar en la demo:

```txt
Nombre real de la marca
WhatsApp real
Instagram / Facebook / Marketplace
Ciudad o zona de entrega
Fotos reales de productos
Lista de productos
Precios
Stock
Métodos de pago
Política de envío o retiro
Testimonios si existen
```

## Advertencia de lenguaje

Evitar promesas médicas o curativas. Usar lenguaje de intención, bienestar, simbolismo, decoración energética y acompañamiento personal.
