## Publicación en Vercel

Para publicarlo como proyecto independiente en Vercel:

1. New Project
2. Importar repo:

```txt
alcivarconsultoresec-lang/test-repo-2115
```

3. En Project Settings / Build & Output Settings configurar:

```txt
Root Directory: clientes/cuarzos/maqueta
Framework Preset: Other
Build Command: vacío
Output Directory: ./
Install Command: vacío
```

4. Deploy.

## Resultado esperado

Un enlace independiente tipo:
