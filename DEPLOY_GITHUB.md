# Despliegue en GitHub Pages

Este repositorio está preparado como sitio estático.

## Activación

1. Entrar al repositorio en GitHub.
2. Ir a `Settings > Pages`.
3. En `Build and deployment`, seleccionar `Deploy from a branch`.
4. Elegir branch `main` y carpeta `/root`.
5. Guardar.

La URL esperada será similar a:

`https://alcivarconsultoresec-lang.github.io/test-repo-2115/`

## Archivos principales

- `index.html`
- `assets/styles.css`
- `assets/app.js`
- `demos/*/index.html`

## Nota

Las demos son conceptuales y ficticias para evitar uso de logos, fotografías o textos de prospectos reales.
