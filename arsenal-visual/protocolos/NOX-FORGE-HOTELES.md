# NOX FORGE / Hoteles

## Uso

Aplicar a hoteles, lodges, hosterías, resorts, espacios boutique y hoteles anfitriones de reuniones, eventos empresariales o Círculo de Gerentes.

## Objetivo

Construir una propuesta digital sobria, institucional y comercialmente persuasiva, mostrando por qué el hotel debe actuar como anfitrión o por qué el huésped debe reservar.

## Estructura obligatoria para hotel anfitrión

1. Hero institucional.
2. Rol del hotel como anfitrión.
3. Beneficios para el hotel.
4. Perfil de asistentes.
5. Formato del evento o reunión.
6. Equipo técnico/comercial detrás.
7. Caso piloto o caso de éxito esperado.
8. Agenda tentativa.
9. CTA reunión/café/visita.
10. WhatsApp o correo de coordinación.

## Estructura para hotel comercial

1. Hero experiencia.
2. Habitaciones o espacios.
3. Experiencias.
4. Restaurante / servicios.
5. Ubicación.
6. Beneficios de reservar directo.
7. CTA reserva / WhatsApp.

## Estética recomendada

Institucional premium + lujo sobrio + confianza ejecutiva.

## Paleta sugerida

- Fondo profundo: #07131E
- Azul petróleo: #0E2A3A
- Oro: #C9A45C
- Marfil: #F4EFE6
- Gris elegante: #B8B1A4

## Tipografías sugeridas

- Playfair Display
- Cormorant Garamond
- Fraunces
- Manrope
- Satoshi

## Errores a evitar

- Parecer folleto turístico genérico.
- No explicar beneficio para el hotel.
- No mostrar el rol de anfitrión.
- Usar lenguaje demasiado informal.
- No dejar claro el siguiente paso.

## Criterio de aprobación

La página debe transmitir prestigio, oportunidad comercial y seguridad operativa.
