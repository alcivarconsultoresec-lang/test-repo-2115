# Patrón: Catálogo Boutique de Cuarzos

## Objetivo

Crear una vitrina digital para negocios de cuarzos, minerales, joyería energética y productos místicos, enfocada en conversión por WhatsApp.

La página no debe sentirse como catálogo genérico. Debe sentirse como:

```txt
joyería energética + boutique espiritual + guía de elección + compra directa por WhatsApp
```

## Cuándo usarlo

- Emprendimientos de cuarzos.
- Tiendas de minerales.
- Joyería energética.
- Sahumerios, péndulos, velas, tarot o productos holísticos.
- Catálogos por intención o energía.

## Estructura recomendada

```txt
Hero boutique
Categorías por intención
Productos destacados
Guía rápida para elegir cuarzo
Packs sugeridos
Galería de piezas reales
Confianza: entregas, pagos, envíos
FAQ
CTA WhatsApp
```

## Categorías por intención

```txt
Protección
Amor propio
Abundancia
Calma
Limpieza energética
Meditación
Regalo especial
Decoración del hogar
```

## Card de producto

Cada card debe incluir:

```txt
Foto
Nombre del cuarzo
Intención principal
Descripción breve
Precio
Disponibilidad
Botón: Quiero este cuarzo
```

Ejemplo:

```txt
Amatista natural
Calma · intuición · descanso
Ideal para meditación, dormitorio o momentos de ansiedad.
Desde $8.000
Consultar disponibilidad por WhatsApp
```

## Guía rápida

Mini cuestionario:

```txt
¿Qué estás buscando hoy?
[Protección]
[Amor]
[Abundancia]
[Calma]
[Limpieza]
[Regalo]
```

Resultado:

```txt
Para protección: obsidiana, turmalina negra, ojo de tigre.
Para amor: cuarzo rosa, rodocrosita.
Para abundancia: citrino, pirita, aventurina.
Para calma: amatista, sodalita, cuarzo blanco.
```

## Estética visual

Paleta recomendada:

```txt
Fondo: blanco perla, lavanda oscuro o arena suave
Acentos: amatista, dorado, cuarzo rosa, verde jade
Texto: grafito, marrón profundo o blanco cálido
Efectos: brillos suaves, partículas, halos, blur, vidrio esmerilado
```

## Componentes recomendados del arsenal

- Magic UI: partículas, beams suaves, hero visual.
- React Bits: efectos de texto y fondos suaves.
- Motion Primitives: transiciones elegantes.
- React Three Fiber + Drei: cristales 3D si el proyecto escala.
- WhatsApp CTA: compra directa por producto.


## Riesgos a evitar

1. No hacer una página demasiado esotérica si el cliente vende de forma más comercial.
2. No prometer curaciones médicas.
3. Usar lenguaje de bienestar, intención y acompañamiento, no afirmaciones absolutas.
4. No mezclar demasiados colores brillantes.
5. No depender solo de fotos genéricas: reemplazar por fotos reales lo antes posible.
