# Patrón: Hero Premium

## Objetivo

Crear la primera sección de una landing con impacto inmediato, claridad comercial y estética profesional. El hero debe vender en menos de 5 segundos qué ofrece el negocio, por qué importa y qué acción debe tomar el usuario.

## Cuándo usarlo

- Landing de cliente premium.
- Demo para prospecto caliente.
- Página de eventos, gimnasio, restaurante, cuarzos, hotel o consultoría.
- Portada de portfolio comercial.

## Estructura recomendada

```txt
[Etiqueta de contexto]
[Título grande]
[Subtítulo con beneficio]
[Botón principal WhatsApp]
[Botón secundario: ver productos / ver fotos / cotizar]
[Badges de confianza]
[Imagen o escena visual fuerte]
```

## Variantes visuales

### Oscuro premium

Usar para:

- gimnasios,
- clubes nocturnos,
- productos tecnológicos,
- CRM,
- servicios de alto impacto.

Características:

- fondo negro o azul profundo,
- gradientes sutiles,
- brillos/neones,
- glassmorphism,
- sombras fuertes,
- titulares enormes.

### Boutique energética

Usar para:

- cuarzos,
- joyería,
- wellness,
- terapias,
- regalos.

Características:

- fondo perla, amatista, arena o lavanda,
- partículas suaves,
- luz difusa,
- textura mineral,
- tipografía elegante,
- tarjetas con brillo sutil.

### Eventos y salones

Usar para:

- Club Montt,
- centros de eventos,
- restaurantes con eventos,
- arriendos.

Características:

- foto amplia del espacio,
- CTA de cotización,
- selector por tipo de evento,
- prueba social,
- badges de capacidad, ubicación y disponibilidad.

## Componentes que pueden inspirar el patrón

- Magic UI: efectos hero, beams y fondos.
- React Bits: textos animados y backgrounds.
- GSAP: entradas y transiciones cinemáticas.
- Lenis: scroll premium.
- shadcn/ui: botones, badges y cards.

## Reglas comerciales

1. El titular debe hablar del resultado, no solo del negocio.
2. El CTA principal debe abrir WhatsApp con mensaje prellenado.
3. No saturar el hero con demasiadas secciones.
4. La imagen debe vender el deseo principal.
5. En móvil, el botón principal debe aparecer visible sin hacer demasiado scroll.

## Prompt base para Codex

```txt
Construye una sección hero premium para [tipo de negocio].
Debe verse moderna, comercial y profesional.
Incluye: etiqueta superior, título fuerte, subtítulo de beneficio, CTA principal a WhatsApp, CTA secundario, badges de confianza y una composición visual de alto impacto.
Usa HTML/CSS/JS puro, responsive, sin dependencias externas.
La estética debe ser [oscura premium / boutique energética / eventos elegante].
```

## Aplicaciones directas

- Metalpunk: hero oscuro, agresivo, con foto real del gimnasio.
- Club Montt: hero de eventos con cotización rápida.
- Cuarzos: hero boutique energético con productos destacados.
- CRM: hero tipo command center con métricas y animaciones.
