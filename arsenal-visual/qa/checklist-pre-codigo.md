# Checklist pre-código NOX FORGE

Antes de programar una página nueva, confirmar:

## Contexto

- [ ] Prospecto identificado.
- [ ] Nicho seleccionado.
- [ ] Dolor comercial definido.
- [ ] Oferta principal definida.
- [ ] Acción de conversión definida.

## Investigación

- [ ] Arsenal visual consultado.
- [ ] Patrones relevantes revisados.
- [ ] Referencias visuales definidas.
- [ ] Assets disponibles revisados.
- [ ] Repositorio del proyecto revisado si existe.

## Diseño

- [ ] Design Canvas preparado.
- [ ] Paleta definida.
- [ ] Tipografía definida.
- [ ] Layout definido.
- [ ] Componentes definidos.
- [ ] Efectos seleccionados.

## Aprobación

- [ ] Usuario aprobó diseño o autorizó programar directo.

## Riesgos

- [ ] No hay placeholders visibles planificados para cliente.
- [ ] No se están copiando páginas anteriores sin adaptación.
- [ ] No se destruirá arquitectura existente.
