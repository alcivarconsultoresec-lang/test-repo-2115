# Checklist Vercel NOX FORGE

Antes de enviar un enlace a cliente:

## Publicación

- [ ] Proyecto desplegado en Vercel.
- [ ] URL correcta identificada.
- [ ] URL abierta y revisada.
- [ ] Se usó parámetro `?v=commit` si hay riesgo de caché.

## Mobile

- [ ] Hero visible en móvil.
- [ ] CTA visible arriba.
- [ ] Dock o botón flotante no tapa contenido.
- [ ] Galerías y cards se adaptan.

## Funcionalidad

- [ ] WhatsApp abre correctamente.
- [ ] Mensaje está prellenado.
- [ ] Formulario funciona o redirige.
- [ ] Mapa abre correctamente.
- [ ] Assets cargan.

## Calidad visual

- [ ] No hay placeholders.
- [ ] No hay textos internos.
- [ ] Imágenes no están tapadas por texto.
- [ ] Contraste correcto.
- [ ] Animaciones funcionan.

## Entrega

- [ ] Enlace registrado en Matriz_Entregas.
- [ ] Mensaje comercial preparado.
- [ ] Próxima acción definida.
