# Checklist WhatsApp NOX FORGE

Antes de entregar una página con WhatsApp:

## Número

- [ ] El número existe.
- [ ] El número parece móvil.
- [ ] En Chile, priorizar formato +56 9.
- [ ] No convertir teléfono fijo sin advertir.

## Mensaje

- [ ] Mensaje prellenado natural.
- [ ] No demasiado largo.
- [ ] Incluye intención.
- [ ] Incluye contexto de origen si aporta.
- [ ] No suena robótico.

## CTA

- [ ] Hay CTA arriba del fold.
- [ ] Hay CTA por producto/plan/servicio si corresponde.
- [ ] Hay botón móvil persistente.
- [ ] El botón abre wa.me correctamente.

## Nicho

- [ ] Eventos: pedir fecha, personas y tipo de evento.
- [ ] Feria: diferenciar visitante/expositor.
- [ ] Gimnasio: diferenciar plan.
- [ ] Mayorista: preguntar categoría/stock.
- [ ] Hotel: pedir reunión/fecha/cargo.

## Prueba

- [ ] Link probado en móvil.
- [ ] Mensaje codificado correctamente.
- [ ] No hay espacios raros ni caracteres rotos.
