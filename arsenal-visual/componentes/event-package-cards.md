# Componente: Event Package Cards

## Objetivo

Presentar paquetes de eventos de forma clara, comparable y accionable.

## Cuándo usarlo

- Salones.
- Clubes.
- Banquetería.
- Productoras.
- Hoteles con eventos.

## Estructura de cada card

- Nombre del paquete.
- Tipo de evento ideal.
- Capacidad sugerida.
- Incluye.
- Extras opcionales.
- Desde / consultar precio.
- CTA WhatsApp.

## Ejemplo de paquetes

- Arriendo simple.
- Cumpleaños familiar.
- Evento empresa.
- Celebración premium.
- Evento completo con producción.

## Reglas UX

- Máximo 3 a 5 paquetes visibles.
- Destacar uno como recomendado.
- No saturar con listas largas.
- CTA claro por paquete.

## Snippet conceptual

```html
<article class="package-card featured">
  <span class="tag">Recomendado</span>
  <h3>Evento privado</h3>
  <p>Ideal para cumpleaños, celebraciones familiares y reuniones privadas.</p>
  <ul>
    <li>Espacio reservado</li>
    <li>Configuración de mesas</li>
    <li>Apoyo de coordinación</li>
  </ul>
  <a class="btn" href="#">Cotizar este paquete</a>
</article>
```

## Checklist

- Se comparan fácil.
- Hay recomendado.
- CTA por paquete.
- No promete servicios no confirmados.
