# Tokens de tipografía NOX FORGE

## Regla general

La tipografía debe responder al nicho. No repetir siempre la misma familia.

## Escala responsive recomendada

```css
--text-xs: clamp(0.72rem, 0.68rem + 0.2vw, 0.82rem);
--text-sm: clamp(0.85rem, 0.78rem + 0.3vw, 0.95rem);
--text-base: clamp(1rem, 0.95rem + 0.35vw, 1.12rem);
--text-lg: clamp(1.15rem, 1rem + 0.7vw, 1.45rem);
--text-xl: clamp(1.5rem, 1.15rem + 1.4vw, 2.2rem);
--text-hero: clamp(3rem, 9vw, 8rem);
```
