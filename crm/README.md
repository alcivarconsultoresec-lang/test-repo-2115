# NØX CRM Command Center

Prototipo visual de CRM comercial para Alcívar Consultores.

## Objetivo

Convertir el seguimiento comercial en una experiencia visual tipo command center:

- prospectos,
- pipeline,
- demos enviadas,
- urgencia comercial,
- seguimiento WhatsApp,
- base de oportunidades,
- priorización asistida por IA.

## URL local dentro del sitio

```txt
/crm
```

## Archivos principales

```txt
crm/index.html
vercel.json
```

## Despliegue recomendado en Vercel

1. Importar el repositorio en Vercel:

```txt
alcivarconsultoresec-lang/test-repo-2115
```

2. Framework preset:

```txt
Other
```

3. Build command:

```txt
None
```

4. Output directory:

```txt
.
```

5. Ruta del CRM después del deploy:

```txt
https://<proyecto-vercel>.vercel.app/crm
```
